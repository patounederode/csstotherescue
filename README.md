# EINDREFLECTIE
Tijdens dit project heb ik veel nieuwe CSS-technieken geleerd, zoals container (style) queries, Has(input:checked), animaties en werken met kleur en gradients in oklch en hsl. Het was in het begin lastig om een goed concept te bedenken, maar uiteindelijk ben ik tevreden met het resultaat.

Ik had graag nog meer aandacht besteed aan details en extra interacties, maar door tijdgebrek was dat niet mogelijk. Feedback vragen heeft me geholpen om sneller vooruit te komen en mijn project verder te verbeteren.

Door dit project ben ik vooral gegroeid in styling en interactie. Ik kan verbeteren door minder te blijven hangen in kleine details, maar over het geheel ben ik tevreden met mijn groei en mijn eindresultaat.
